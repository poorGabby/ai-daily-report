# 设计趋势 - 2026-04-05

> AI交互方式、Prompt UX、Agent UX、Workflow

## 今日概览

今日设计趋势领域重点关注代理式UX和ChatGPT App SDK。代理式UX通过AI代理与用户协作，革新设计方法；而ChatGPT App SDK允许开发者在聊天中构建应用，为用户体验设计带来新可能。这两个动态最值得关注，预示着设计领域的AI融合趋势。

**一句话摘要**: 代理式UX是一种AI代理与用户共同决策和协作的设计方法。

**能解决什么问题**: 解决传统设计中缺乏个性化和主动性的问题。

**产品应用**: 可应用于智能助手和自动化服务，提升用户体验和效率。

**示例场景**: 智能助手根据用户习惯自动调整日程安排。
