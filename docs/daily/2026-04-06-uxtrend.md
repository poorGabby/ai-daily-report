# 设计趋势 - 2026-04-06

> AI交互方式、Prompt UX、Agent UX、Workflow

## 今日概览

今日设计趋势资讯汇总：代理式UX的兴起正在改变设计领域的传统认知，AI代理与用户共同决策和协作成为新焦点。AI代理工作流设计模式的探讨揭示了其分类和应用实例。产品设计师开始将AI工具融入工作流程，面临应用与挑战。提示设计的最佳实践强调简洁、明确、具体。AI/UX领域正从以人为中心的设计转向包含智能代理的设计。最值得关注的是代理式UX对设计认知的影响以及AI代理工作流设计模式的分类和应用。
