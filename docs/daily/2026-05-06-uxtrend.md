# 设计趋势 - 2026-05-06

> AI交互方式、Prompt UX、Agent UX、Workflow

## 今日概览

今日设计趋势领域聚焦于AI技术在工作流优化和用户体验提升的应用。主要动态包括：AI工具提升设计效率、AI交互模式增强用户互动、AI辅助用户保持专注、产品设计师融入AI工具以及探讨基于LLM的AI代理工作流设计模式。最值得关注的是AI工作流优化技巧和AI交互模式，它们直接影响设计效率和用户体验。
