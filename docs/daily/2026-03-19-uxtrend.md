# 设计趋势 - 2026-03-19

> AI交互方式、Prompt UX、Agent UX、Workflow

## 今日概览

今日设计趋势聚焦于人工智能代理的用户体验。主要动态包括：1) 代理性用户体验设计指南强调理解用户目标和偏好的重要性；2) AI交互模式的探索，旨在让用户轻松识别并互动AI功能；3) 从人为中心的设计转变为包含智能代理的设计；4) ChatGPT的UI/UX重新设计案例研究，提升用户体验；5) 设计AI体验时需考虑的指导原则。最值得关注的是AI交互模式的探索和设计AI体验的指导原则。
