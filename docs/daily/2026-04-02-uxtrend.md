# 设计趋势 - 2026-04-02

> AI交互方式、Prompt UX、Agent UX、Workflow

## 今日概览

今日设计趋势领域主要聚焦于AI技术在用户体验设计中的应用。AI驱动的用户体验设计代理能够集成干系人意见，自动迭代优化设计，并生成文档。同时，AI/UX领域强调了人工智能代理在体验设计中的角色转变，从以人为中心的设计转向包含代理的设计。最值得关注的是AI代理在产品设计中的快速独立能力以及ChatGPT在用户界面设计中的应用，这标志着AI技术与设计领域的深度融合。
