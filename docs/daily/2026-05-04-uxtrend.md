# 设计趋势 - 2026-05-04

> AI交互方式、Prompt UX、Agent UX、Workflow

## 今日概览

今日设计趋势聚焦于AI技术对设计领域的深远影响。2026年，AI将重塑设计流程和决策权，同时深度融入工作流程，提升创造力和协作效率。值得关注的是，AI时代的美学将通过视觉和网页设计展现新趋势，而产品设计中AI的应用将快速实现概念到原型的转化。
