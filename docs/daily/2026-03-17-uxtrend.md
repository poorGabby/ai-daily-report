# 设计趋势 - 2026-03-17

> AI交互方式、Prompt UX、Agent UX、Workflow

## 今日概览

今日设计趋势聚焦于AI交互与工作流程优化。AI交互模式探索人机互动新方式，提升用户体验；AI工作流程设计模式结合AI与机器学习自动化任务执行。最值得关注的是AI界面设计的7个关键模式，以及产品设计师如何将AI融入设计工作流程，这些趋势预示着设计领域向智能化的进一步发展。
